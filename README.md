## 📖 Overview

DataMind AI is an intelligent data analysis application designed to simplify exploratory data analysis (EDA) for Excel and CSV datasets.

Instead of manually inspecting spreadsheets, users can upload a dataset and instantly receive:

- Dataset profiling
- Interactive dashboard
- Data quality assessment
- Statistical summaries
- Automated data cleaning suggestions
- AI-powered chat for asking questions about the data
- Exportable analysis results

The project demonstrates practical applications of AI in data analytics while showcasing modern frontend development using React and TypeScript.

# ✨ Features

## 📂 Data Upload

- Upload CSV files
- Instant dataset preview
- Row & column statistics
- Search records

---

## 📊 Data Profile

Automatically analyzes the dataset and displays:

- Number of rows
- Number of columns
- Data types
- Missing values
- Duplicate rows
- Memory usage
- Numerical summaries

---

## 📈 Interactive Dashboard

Visualize your data using interactive charts:

- Histograms
- Bar charts
- Pie charts
- Distribution analysis
- Top categories
- Location analysis
- Industry analysis

---

## 🧹 Data Cleaning

Automatically detect:

- Missing values
- Duplicate records
- Inconsistent formatting
- Invalid values

Cleaning actions include:

- Remove duplicates
- Fill missing values
- Standardize formats
- Normalize columns

---

## 📐 Statistical Analysis

Generate descriptive statistics including:

- Mean
- Median
- Standard deviation
- Min / Max
- Quartiles
- Distribution metrics

---

## 🤖 AI Chat Assistant

Ask questions in natural language, such as:

> Which industry has the most layoffs?

> Show companies with the highest funding.

> Summarize this dataset.

> Which columns contain missing values?

The AI interprets your request and provides insights without requiring SQL or Python.

---

## 📤 Export Results

Export processed datasets and analysis results for further reporting.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/DataMind-AI.git

cd DataMind-AI
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open

```
http://localhost:5173
```
